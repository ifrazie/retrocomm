# Product Overview

**Retro Messenger** is a nostalgic web application that reimagines modern messaging through the lens of 1980s-90s communication technology. The app provides an immersive retro experience with two distinct interface modes:

## Core Concept

Transform contemporary messaging into an engaging experience that feels like using vintage pagers and fax machines, while maintaining full modern functionality including:
- Real-time multi-user messaging
- AI-powered chatbot integration (via LM Studio)
- End-to-end encryption (RSA-2048 + AES-256-GCM)
- Webhook integration for external services
- Password-protected user accounts

## Interface Modes

### Pager Mode
Classic green-on-black LCD display with authentic pager interface, physical button controls, and CRT screen effects. Displays the last 5 messages with structured formatting.

### Fax Mode
Vintage thermal fax machine interface with dot-matrix printer styling, progressive transmission animation, and full message history displayed as fax documents.

## Key Features

- **Dual retro interfaces** (pager/fax) with authentic aesthetics
- **AI chatbot** powered by LM Studio with context-aware conversations
- **Security-first** with E2EE, bcrypt password hashing, and zero-knowledge architecture
- **Multi-user support** with real-time messaging via Server-Sent Events (SSE)
- **Webhook integration** for external service connectivity
- **Responsive design** optimized for desktop and mobile

## Target Audience

Developers and tech enthusiasts who appreciate retro aesthetics, nostalgic design, and innovative approaches to familiar applications. Built for the Kiroween Hackathon 2025 to showcase webhooks and chatbot integration.
